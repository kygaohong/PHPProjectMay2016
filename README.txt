This website is built with HTML, CSS and PHP by Hong Gao. The aim of this project is to meet the requirement
of PHP Development of CodeLouisville May 2016 session. 
A sql database was cretaed and used in this project. The website is linked to the sql database. 
Users can dynamically interact with the database by typing a book name and a minimum age for this book in the website. 
A sql file gift.sql was included and can be imported in phpMyadmin. 
Media Queries were used to make this website responsive for different device sizes.
